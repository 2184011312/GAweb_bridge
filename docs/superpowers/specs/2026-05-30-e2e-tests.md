# Arc Tunnel — E2E Test Suite

**Date:** 2026-05-30  
**Method:** Self-bootstrapping — AI calls arc-tunnel MCP tools on real browser tabs and validates responses.

## Why No Test Framework

Jest cannot test extension-side code (chrome.debugger, chrome.tabs, chrome.storage) — these browser APIs don't exist in Node. The MCP server side (command-queue, types, tools, websocket-server) already has 14 unit tests.

The most realistic test is AI directly invoking MCP tools against a live browser — the same path real users take.

## Test Plan

### Prerequisites
- Arc Tunnel extension loaded and connected in browser
- At least one tab open (any URL)
- Claude Code with arc-tunnel MCP server connected

### Test Cases (15 tools)

| # | Tool | Action | Expected |
|---|------|--------|----------|
| 1 | `list_tabs` | Call without params | Returns `{ tabs: [...] }` with ≥1 tabs |
| 2 | `navigate` | `{ tabId, action: "goto", url: "https://example.com" }` | `{ status: "navigated" }` |
| 3 | `snapshot` | `{ tabId }` on example.com | Returns tree with at least "Learn more" link |
| 4 | `interact` | `{ tabId, action: "click", target: "e1" }` | `{ status: "click" }`, page navigates |
| 5 | `execute_script` | `{ tabId, script: "document.title" }` | Returns `{ result: "Example Domain" }` |
| 6 | `screenshot` | `{ tabId }` | Returns base64 PNG data (iVBOR header) |
| 7 | `start_recording` | `{ tabId }` | Returns `{ recordingId: "..." }` |
| 8 | `stop_recording` | `{ tabId }` | Returns `{ recording: { actions: [...] } }` |
| 9 | `replay_recording` | `{ recordingId, tabId }` | `{ status: "replayed" }` |
| 10 | `create_tab` | `{ url: "https://example.com" }` | Returns `{ tabId: number }` |
| 11 | `close_tab` | `{ tabId }` | `{ status: "closed" }` |
| 12 | `get_console_logs` | `{ tabId }` | Returns `{ logs: [...] }` |
| 13 | `manage_storage` | `{ tabId, type: "cookie", action: "list" }` | Returns `{ cookies: [...] }` |
| 14 | `save_session` | `{ name: "e2e-test" }` | Returns `{ sessionId: "..." }` |
| 15 | `restore_session` | `{ sessionId }` | `{ status: "restored" }` |

## Manual Execution
